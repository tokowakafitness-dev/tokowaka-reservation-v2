システムの全体像
```
TRAINER'S OVERVIEW — あなたに関係する部分だけ
📅
あなたがやること
TOKOWAKA用カレンダーに「出勤可能」を入力するだけ。それ以外は全て自動。
⚙️
システムが自動でやること
出勤可能な時間帯を予約サイトに表示 → 予約が入ったらカレンダーに追加・メール送信。
⚠️
入れ忘れた場合
その時間帯は予約サイトに表示されないだけ。誤予約は入りません。
予約サイト：予約サイトを確認する →
反映タイミング：入力後、最大5分で反映されます
```
1
新規カウンセリング予約
```
出勤可能枠の登録（毎週やること）
1
Googleカレンダーアプリを開く
2
「自分の名前（TOKOWAKA用）」カレンダーで予定を作成
3
タイトルを「出勤可能」と入力し、開始・終了時間を設定して保存
📅 表示期間：直近2週間分が表示されます。週の始めにまとめて入力してください
🔒 プライベートカレンダーは不要：TOKOWAKA用カレンダーのみ操作します
予約が入ったら（自動 → あなたの対応）
自
カレンダーに予約イベントが追加・招待メールが届く
1
招待メールを確認し「承認」する
2
当日、予約時間の15分前までに来店
3
カウンセリング実施（75分・無料）→ 入会意向を確認
⚠️ 急なキャンセル・変更は龍之介（中野）に連絡してください
```
2
通常予約の管理
```
カレンダーの使い分け
予約の種類地下1階
既存顧客トレーニング・レンタル✓ 入力不要
会議・商談・イメコン不要✓ 入力
新規カウンセリング自動自動
タイトルは何でも構いません。入力した時間帯は全て予約枠としてブロックされます
予約の命名規則
内容 _ 担当者名 _ 顧客名
例：トレーニング_中野_山田様　／　会議_中野_（相手先）
なぜ必要か：毎月の給与計算のため、担当者ごとの件数をスプレッドシートで集計しています。規則通りに入力されていないと集計できません。
給与計算の流れ（現状・改善予定）
1
毎月末に中野がカレンダーデータをスプレッドシートに集計
2
担当者名をもとにトレーナーごとの件数・時間を集計
3
報酬計算（月8回コース：¥10,900/回）→ 支払い
```
3
注意事項・よくある質問
```
出勤可能を入れ忘れた場合
予約サイトに表示されないだけです。誤予約は入りません。気づいたタイミングで入力してください。
急に出勤できなくなった場合
「出勤可能」の予定を削除してください。すでに予約が入っている場合は龍之介（中野）に連絡してください。
プライバシーについて
「TOKOWAKA用」カレンダーのみ共有しています。プライベートの予定は一切外部から見えません。
```

不明点は 龍之介（中野） までご連絡ください
